# Eden

Projekt, který pracuje s mapou, pojmem, pojmenováváním. Základem je mapa obsahující pouze krajinné prvky, jako jsou vodní toky a plochy, lesy, louky a travnaté plochy, ledovce, skály, pouště a další. Názvy míst nebo krajinných prvků, jsou v mapě nahrazeny kombinací dvou znaků, minimem pro to něco pojmenovat. Často metaforické názvy jako je například Divoký potok z mapy mizí a jsou nahrazeny strojem přiřazeným daleko utilitárnějsím názvem připomínajícím nějaké ID.
